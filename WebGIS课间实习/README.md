# 武汉大学遥感学院 网络GIS 课间实习作业

地理信息购物平台　实现如下功能：

* 利用Vue+ElementUI设计网站布局
* 利用Vuex实现满足地图数据服务的产品订购页面（购物车）
* 体验一种基于商业GIS、开源GIS或网络地图API的入门或部分功能
* AJAX技术实现前后端数据交互
* 利用HTML5画布Canvas实现专题地图（统计图表）的绘制（Echarts）
